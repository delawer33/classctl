УСТАНОВКА CLASSCTL
==================


LINUX
-----

1. Установите pipx (если ещё не установлен):

   sudo apt install pipx
   pipx ensurepath

   После этого откройте новый терминал (или выполните: source ~/.bashrc).

2. Установите classctl:

   pipx install git+https://github.com/delawer33/classctl

3. Выдайте права на ARP-сканирование (один раз после установки):

   sudo setcap cap_net_raw+ep "$(find ~/.local/share/pipx/venvs/classctl -name python3 -type f | head -1)"

4. Запустите:

   classctl

   Откройте браузер: http://127.0.0.1:8000

ВАЖНО: при обновлении Python (apt upgrade) или classctl (pipx install ... --force)
шаг 3 нужно повторить — обновление пересоздаёт виртуальное окружение и сбрасывает права.


WINDOWS
-------

1. Установите Npcap: https://npcap.com/#download
   (нужен только для Wake-on-LAN; обнаружение машин работает без него)

2. Установите pipx:

   pip install pipx
   pipx ensurepath

   Перезапустите терминал.

3. Установите classctl:

   pipx install git+https://github.com/delawer33/classctl

4. Запустите:

   classctl

   Откройте браузер: http://127.0.0.1:8000


ОБНОВЛЕНИЕ
----------

   pipx install git+https://github.com/delawer33/classctl --force

   Linux: после обновления повторите шаг 3 (setcap).
